Usability Test Results


Website that was tested

Book Collection App


User Demographics

20 year old, Female, College Student, Accounting Major


Tasks

Register for a new account

User was quick to find the link to click for registering a new account as it is written clearly on the login page.

Upon clicking, the user found the form very easy to fill out as there were labels for everything.

No successful registration message, just goes back to login page.


Login to the site with your new account

User was already sent back to the login page, so she easily was able to fill out the form to login and successfully login.

After logging in, got sent right to home page of the app.


Search for books by George Orwell

User saw the drop down next to the search bar and immediately knew that was the search type, so she selected Author to search by author.

Then easily filled out the form to search by typing in "George Orwell" and the results came up successfully.


Add 3 books to your favorites

User was hesitant as there is nothing on the screen that explicitly says "Add to favorites".

Guessed that you have to click on a book to add it to favorites, and when she clicked a popup came up and that's where she found out how to add to favorites.

User was greeted kindly with a success message after adding to favorites, and easily repeated this 2 more times.


View your favorites and delete one

There is no favorites tab, only a "My Collection" tab, so the user was a bit confused at first.

The user assumed that "My Collection" contained favorites, so she clicked that and successfully went to view her favorites.

User felt that the list was a bit unorganized since there is no visual separation between book information.

User easily removed a book from favorites by clicking the "Remove from favorites" button, but felt it wasn't the easiest to tell which book she was deleting.

User liked that there was a warning asking if you are sure you want to delete the book, and that there was a success message.


Log out of the app

User assumed from past experience that the logout button might be in the top right, which it was.

User then easily clicked on the logout link and it logged her out successfully.


Improvements

Usability

Changed wording from "My Collection" to "My Favorites"

Added lines between books on favorites list

Added text on the homepage with some guidance on how to use the app

Added an underline to forgot password and register here so you know where to click/know what's clickable


Stylistic

Styling choice just wanted to make all the buttons match
